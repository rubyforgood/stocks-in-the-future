# Security Policy

## Supported Versions

Only the latest version [main branch](https://github.com/rubyforgood/stocks-in-the-future) and currently deployed version of this project is in scope for security issues. 

Also, only the production environment is in scope, although, it's ok and normal to test the staging environment.

## Reporting a Vulnerability

Please report a vulnerability by emailing [casa@rubyforgood.org](mailto:polly@rubyforgood.org)

We will reply to all reported issues.

We do not have any bug bounty program but we will be happy to list your name in our contributors list! :)
